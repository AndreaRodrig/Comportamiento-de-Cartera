# Comportamiento-de-Cartera
Modelo predictivo de cobranza para clientes de una empresa retail, con el objetivo de identificar clientes que probablemente se vuelvan “malos” en el próximo mes. El proyecto se centra en la predicción de t → t+1, es decir, el target se calcula para el próximo mes, asegurando validez temporal y evitando fugas de información futura.
